# Debug Keys

These are the intended development-only controls for the current milestone:

- `F1`
  Toggle debug overlay
- `F2`
  Print current run / floor / seed
- `F5`
  Regenerate current floor with the same seed
- `Shift+F5`
  Regenerate current floor with a new seed
- `F6`
  Teleport to entry
- `F7`
  Teleport to stairs down
- `F8`
  Teleport to landmark
- `F9`
  Teleport to secret
- `F10`
  Return safely to `TownHub`

Additions for later gates:
- kill all enemies on the current floor
- grant small debug test resources if needed
