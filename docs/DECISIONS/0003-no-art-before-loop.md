# Decision 0003: No Art Before the Loop Is Proven

## Status
Accepted

## Context
The game has a technical foundation but does not yet have a proven fun loop.

## Decision
Do not import Fab assets or start a real art pass during this milestone.

## Why
- prevents avoiding gameplay truth with decoration
- reduces render/material churn while the game is still graybox
- keeps focus on geometry, combat, rewards, and exploration hooks

## Consequences
- town and dungeon remain graybox for this milestone
- art intake starts only after the loop is proven fun and stable
